# HurraS2SSDK

A Swift package for iOS applications that enables server-to-server (S2S) tracking of user events and views.

## Overview

HurraS2SSDK provides a simple interface for sending event data to Hurra's analytics backend while respecting user privacy preferences. It supports tracking page views, user interactions, and custom events with minimal setup.

## Documentation

- [Privacy Preferences API Documentation](PrivacyPrefsAPI.md) - Detailed documentation for managing user privacy preferences and consent settings

## Installation

### Swift Package Manager

Add the package to your `Package.swift`:

```swift
dependencies: [
    .package(url: "https://github.com/yourusername/HurraS2SSDK.git", from: "1.0.0")
]
```

Or add it directly in Xcode:
1. File > Add Packages...
2. Enter the repository URL: `https://github.com/yourusername/HurraS2SSDK.git`
3. Select the version you want to use

## Configuration

Initialize the SDK with your account credentials:

```swift
import HurraS2SSDK

// use user ID generated by SDK
let hurraSDK = HurraS2SSDK(
    accountId: "your_account_id",
    apiKey: "your_api_key"
)

// Or use IDFA for user identification
let hurraSDK = HurraS2SSDK(
    accountId: "your_account_id",
    apiKey: "your_api_key",
    useAdvertiserId: true  // Use IDFA if available
)

// Or use user ID managed by app
let hurraSDK = HurraS2SSDK(
    accountId: "your_account_id",
    apiKey: "your_api_key",
    customUserId: nil       // Provide a custom user ID if needed
)
```

### Storing API Credentials

For security, store your API credentials outside of your source code:

#### Option 1: Using Info.plist

Add to your app's Info.plist:

```xml
<key>HurraAPICredentials</key>
<dict>
    <key>AccountId</key>
    <string>your_account_id</string>
    <key>ApiKey</key>
    <string>your_api_key</string>
</dict>
```

#### Option 2: Using Environment Variables

Set environment variables:

```bash
export HURRA_ACCOUNT_ID=your_account_id
export HURRA_API_KEY=your_api_key
```

## Usage

### Tracking Views

```swift
// Basic page view tracking
Task {
    await hurraSDK.trackView(
        eventData: [
            "page_type": "product_detail",
            "product_id": "12345",
            "category": "electronics"
        ],
        currentView: "https://example.com/products/12345"
    )
}
```

### Tracking Events

```swift
// Track user interactions
Task {
    await hurraSDK.trackEvent(
        eventType: "add_to_cart",
        eventData: [
            "product_id": "12345",
            "price": "99.99",
            "currency": "USD",
            "quantity": 1
        ],
        currentView: "https://example.com/products/12345"
    )
}

// With completion handler
Task {
    await hurraSDK.trackEvent(
        eventType: "purchase",
        eventData: [
            "order_id": "ORD-12345",
            "total": "99.99"
        ],
        currentView: "https://example.com/checkout/confirmation",
        isInteractive: true
    ) { result in
        switch result {
        case .success(let response):
            print("Event tracked successfully: \(response)")
        case .failure(let error):
            print("Failed to track event: \(error)")
        }
    }
}
```

### Managing Privacy Preferences

```swift
// Create privacy preferences
let privacyPrefs = PrivacyPrefs()

// Set all categories
privacyPrefs.setAllCategories(true)

// Or set specific categories
privacyPrefs.setCategory(.essential, accept: true)
privacyPrefs.setCategory(.analytics, accept: true)
privacyPrefs.setCategory(.functional, accept: false)
privacyPrefs.setCategory(.advertisement, accept: false)
privacyPrefs.setCategory(.personalization, accept: false)

// Apply preferences to the SDK
hurraSDK.setPrivacyPrefs(privacyPrefs)
```

### User Identification

```swift
// Get current user ID
let userId = hurraSDK.getUserId()

// Set Advertiser ID (IDFA)
hurraSDK.setAdvertiserId()

// Set custom user ID
hurraSDK.setUserId("custom-user-id")
```

## SwiftUI Integration

```swift
import SwiftUI
import HurraS2SSDK

@main
struct MyApp: App {
    @UIApplicationDelegateAdaptor(AppDelegate.self) var appDelegate
    
    var body: some Scene {
        WindowGroup {
            ContentView()
                .environmentObject(appDelegate.hurraSDK)
        }
    }
}

class AppDelegate: NSObject, UIApplicationDelegate {
    @MainActor let hurraSDK = HurraS2SSDK(
        accountId: "your_account_id",
        apiKey: "your_api_key"
    )
}

struct ContentView: View {
    @EnvironmentObject var hurraSDK: HurraS2SSDK
    
    var body: some View {
        NavigationView {
            Text("Hello, World!")
                .onAppear {
                    Task {
                        await hurraSDK.trackView(
                            eventData: ["page_type": "home"],
                            currentView: "home"
                        )
                    }
                }
        }
    }
}
```

## Response Format

The SDK returns a `ServerResponse` object with the following structure:

```swift
public struct ServerResponse: Codable {
    public let status: Int
    public let message: String?
    public let data: [String: String]?
}
```

- `status`: 1 for success, 0 for accepted but not processed, negative values for errors
- `message`: Optional message from the server
- `data`: Optional additional data from the server

## Thread Safety

The SDK is designed to be thread-safe with `@MainActor` annotations on its public methods. Always call SDK methods from the main thread or use `Task` with `await` to ensure proper execution.

## Privacy Considerations

- The SDK respects user privacy preferences set through the `PrivacyPrefs` class
- By default, it uses a generated UUID for user identification
- IDFA (Advertiser ID) usage is optional and requires user permission
- All data is sent securely over HTTPS
